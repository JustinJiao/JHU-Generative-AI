# Generated AI Homework1

## Description
This repository contains the first assignment for Generative AI. It includes the process of initialising the repository to have Claude generate code, committing the changes, manually modifying and committing the code, and finally adding a README file and committing it. Throughout this process, we can see that the AI is able to generate the required output, although some of the statements are not entirely accurate.

## Git Steps
1. Initialise the repository include assistant_draft.md and final_version.md, and README.md files.And commit the changes with the message "initial commit".
2. Add content to assistant_draft.md using Claude code, and commit the changes with the message "add assistant_draft.md".
3. Manually modify the content in assistant_draft.md to better reflect my intentions, and commit the changes with the message "add final edited version".
4. Add content to README.md, and commit the changes with the message "add README.md".
5. I have amended the README as it contained some spelling mistakes and uploaded it to the repository.
